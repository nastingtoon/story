# Story Text Editor PWA

Это веб-приложение в формате PWA, которое можно открыть на iPhone как отдельное приложение с иконкой на домашнем экране.

## Что умеет
- выбрать фото из галереи
- добавить несколько текстовых слоёв
- редактировать текст по тапу
- менять цвет, размер, выравнивание
- включать тёмную подложку под текст
- двигать текст одним пальцем
- масштабировать и вращать двумя пальцами
- сохранить итоговую картинку как PNG

## Как запустить на iPhone

### Вариант 1 — самый простой через любой хостинг статических файлов
Подойдут:
- Netlify
- Vercel
- GitHub Pages
- любой свой сервер

Нужно загрузить содержимое этой папки как статический сайт:
- index.html
- manifest.json
- sw.js
- Shevko-Regular.ttf
- папка icons

После публикации:
1. Открой ссылку в Safari на iPhone
2. Нажми "Поделиться"
3. Выбери "На экран Домой"
4. Сохрани ярлык
5. Открывай уже с иконки, как отдельное приложение

### Вариант 2 — локально проверить на Mac
В терминале внутри папки можно запустить простой сервер:

```bash
cd story-pwa
python3 -m http.server 8080
```

Потом открыть в браузере:
```bash
http://localhost:8080
```

Для iPhone нужен именно доступный по сети адрес или опубликованный сайт.

## Структура
- `index.html` — всё приложение
- `manifest.json` — настройки PWA
- `sw.js` — офлайн-кэш
- `Shevko-Regular.ttf` — встроенный шрифт
- `icons/` — иконки приложения

## Важное ограничение
На iPhone веб-приложение обычно сохраняет файл через загрузку изображения, а не напрямую в системную галерею как нативное iOS-приложение. После экспорта PNG может открыться как скачанный файл, который потом можно сохранить в Фото вручную.
